# adamabd-razak.github.io

This repository hosts my **online CV website** built with GitHub Pages and the **Minimal** Jekyll theme.

Live site: https://adamabd-razak.github.io

## About

The site is built using Markdown (`index.md`) and configured with `_config.yml`.  
The Jekyll Minimal theme provided by GitHub Pages is used for clean layout and design.

## Contents

- `index.md` — Main CV / homepage
- `_config.yml` — Jekyll configuration and theme setup
